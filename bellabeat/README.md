# Bellabeat Case Study

This project analyzes smart device data to understand user activity patterns and support product strategy for the Bellabeat Leaf. The work includes data cleaning, transformation, visualization, and insight development aligned with the Bellabeat case study requirements.

## What I Did
- Cleaned and combined daily activity data using Excel and Power Query  
- Exported a cleaned dataset for analysis  
- Connected the dataset to Tableau  
- Built three visuals: daily steps trend, activity intensity breakdown, and calories vs. steps  
- Identified key behavioral insights from the data  
- Wrote a concise case study brief summarizing findings and recommendations

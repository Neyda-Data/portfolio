# Latin School SQL Project (UCSD)

## Overview
This project analyzes student, course, faculty, and enrollment data for the Latin School database. It includes data cleaning, staging, importing into SQL Server, building an ERD, and writing SQL queries to explore the dataset.

## Tools Used
- SQL Server
- SSMS
- Excel (initial cleanup)
- Draw.io or Lucidchart (for ERD)

## Folder Structure
- **sql/** – SQL scripts for table creation, staging, queries, and stored procedures
- **erd/** – ERD diagram(s)
- **docs/** – project summary and supporting documents

## Key Deliverables
- ERD diagram
- SQL scripts
- Final project summary
